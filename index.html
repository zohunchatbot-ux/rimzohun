API Documentation

POST /api/saweria/createPayment

Create new Saweria payment and generate QR code

Request Headers:

Content-Type: application/json

Request Body:

Parameter Type Required Description
user_id string Yes Saweria User ID
amount integer Yes Payment amount (min: 1000)
name string Yes Customer name
email string Yes Saweria account email
msg string No Payment message

Example Request:

{ "user_id": "saweria_user_123", "amount": 25000, "name": "John Doe", "email": "john@example.com", "msg": "Payment for premium access" }

Response:

{ "status": "success", "code": 200, "data": { "amount": 25000, "amount_to_display": "Rp 25,000", "created_at": "2024-01-20T10:30:00Z", "id": "pay_123456789", "message": "Payment for premium access", "status": "pending", "expired_at": "2024-01-20T11:30:00Z", "receipt": "https://saweria.co/transaction/pay_123456789", "url": "https://saweria.co/payment/pay_123456789", "qr_image": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAA..." } }
GET /api/saweria/checkPayment/{payment_id}

Check payment status

Example Request:

GET /api/saweria/checkPayment/pay_123456789

Response:

{ "status": "success", "data": { "id": "pay_123456789", "amount": 25000, "status": "paid", "paid_at": "2024-01-20T10:35:00Z" } }

Code Examples

Python Example:

import requests url = "https://rimzohun.vercel.app/api/saweria/createPayment" payload = { "user_id": "your_saweria_user_id", "amount": 25000, "name": "Customer Name", "email": "your_email@saweria.com", "msg": "Payment from bot" } response = requests.post(url, json=payload) if response.status_code == 200: data = response.json() qr_image = data['data']['qr_image'] payment_url = data['data']['url'] print("Payment created successfully!") else: print("Error:", response.text)

cURL Example:

curl -X POST "https://rimzohun.vercel.app/api/saweria/createPayment" \ -H "Content-Type: application/json" \ -d '{ "user_id": "your_saweria_user_id", "amount": 25000, "name": "Customer Name", "email": "your_email@saweria.com", "msg": "Payment from bot" }'
